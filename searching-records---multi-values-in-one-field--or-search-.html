Searching records - multi values in one field (OR search)

1. Open any module to search. Here we opened the Catalog (VP).

We are logged in as Earth Sciences Collections Manager, but we still are able to search across collections.

2. Click the "Collection" field

We will see if we can find all the records included in both the Mineralogy and Entomology collections.

3. Add both search terms in separate lines

You will notice that there should be multiple lines showing in your field. Add each term to a different line to perform an OR search that will include both terms.


Tip: if you keep scrolling on that field you can keep adding more terms individually.


4. Click the "Search" icon

The results should return all the records in the two collections searched for.
